# 📊 Supply Chain Analytics Dashboard

## 🚀 Overview
This project analyzes delivery performance data to identify inefficiencies in supply chain operations.

## 🎯 Key Metrics
- Average Delivery Time
- Percentage of Delayed Orders
- Total Cost

## 📊 Analysis Performed
- Vendor-wise performance analysis
- City-wise delivery analysis
- Delay distribution tracking

## 🛠 Tools Used
- Google Sheets / Excel
- Pivot Tables
- Charts & Dashboard

## 📈 Outcome
Built a dashboard to monitor supply chain KPIs and enable data-driven decision making.
